Tofu.js需要运行环境。在开始我们的第一个项目前，你需要先安装本地运行环境。比如：phpstudy(下载)

环境安装完成后，在环境根目录下，创建tofusite文件夹。完成快速入门后，最终的目录结构如下图所示。

tofusite/

1、加载第一个组件

在tofusite目录下，新建index.html。用一个标签 data-tofu-module='组件名' 指定要加载的组件和插入位置。

tofusite/index.html

<!doctype html>
    <html>
        <head>...</head>
    <body>
        <div data-tofu-module='one'>第一个组件</div>
        <script src='./js/tofu.js'></script>
        <script>
            Tofu.config({
                module_base : '/tofusite/modules'
            });
            Tofu();
        </script>
    </body>
</html>

在tofusite目录下，新建modules文件夹，用于存放我们的组件。

在modules文件夹中创建第一个组件，one。one组件只有一个简单的页面index.html和一个controller.js文件。

modules/one/index.html

<h1>你好，我是第一个组件</h1>
<a id='addtwo'>动态加载第二个组件</a>

modules/one/js/controller.js

Tofu.ready('one',function(){
    this.container.innerHTML = this.template;
});

现在访问 localhost/tofusite，你会看到第一个组件已经被加载到了主页面中。


2、动态加载第二个组件

给第一个组件的a标签添加事件，点击它后加载第二个组件。

修改 modules/one/js/controller.js

Tofu.ready('one',function(){
    this.container.innerHTML = this.template;

    window.addtwo.addEventListener('click',function(){
        var domtow = document.createElement('div');
        domtow.setAttribute('data-tofu-module','two');
        document.body.appendChild(domtow);
        Tofu('two');
    });
});

第二个组件的结构和第一个一样，index.html 和controller.js

modules/two/index.html

<h1>第二个组件</h1>

modules/two/js/controller.js

Tofu.ready('two',function(){
    this.container.innerHTML = this.template;
});

现在访问 localhost/tofusite，点击“动态加载第二个组件”，第二个组件被动态的加入到主页面中。


你现在已经知道Tofu.js是干什么的，以及怎么使用它。更进一步了解Tofu.js，请看文档。
